About Project

What is SmartFAQBot?

An intelligent chatbot system designed to assist students and visitors with college-related and local area queries.

Objective

To develop an intelligent web-based chatbot system that provides quick and accurate answers to college and local area-related queries, reducing manual effort and improving user experience.

Technology Used

Built using HTML5, CSS3, JavaScript for frontend; Python Flask for backend; MySQL for database; and NLP-based logic for intelligent response matching.

Target Users

Students, parents, new admissions, faculty, and local visitors who need quick information about the college campus and surrounding local area services.

Key Benefits

24/7 availability, instant responses, no waiting time, reduced burden on administration staff, and consistent accurate information delivery.
