# SmartFAQBot — Intelligent College FAQ Chatbot

BCA Minor Project | Kumaun University, Nainital | 2024-25

## Live Website
Hosted on GitHub Pages — see repository settings for the URL.

## How to Use
- Open `index.html` for the main website
- Open `admin/admin.html` for the Admin Panel
- Login: **admin** / **admin123**

## Project Structure
```
/
├── index.html          ← Main website (GitHub Pages root)
├── css/
│   └── style.css
├── js/
│   ├── chatbot.js
│   └── script.js
├── admin/
│   ├── admin.html
│   ├── admin.css
│   └── admin.js
├── CNAME               ← Custom domain (edit or delete if not using)
└── README.md
```

## Tech Stack
HTML5 | CSS3 | JavaScript | Python Flask | MySQL

## GitHub Pages Setup
1. Go to Settings → Pages
2. Source: Deploy from branch → main → / (root)
3. Save — your site will be live in 2 minutes
